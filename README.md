# Jadwal ASAS - MI PLUS DARUL FALAH

Aplikasi web untuk menampilkan jadwal ASAS (Asesmen Sumatif Akhir Semester) MI PLUS DARUL FALAH GAMBIRAN MOJOAGUNG.

## Fitur
- Tampilan jadwal semua kelas (1-6)
- Pencarian guru dengan autocomplete
- Highlight jadwal guru yang dicari
- Responsive design untuk semua perangkat
- Search bar sticky yang mudah diakses

## Languages
- HTML5, CSS3, JavaScript vanilla

## Features
- Responsive design
- Mobile-friendly

## Cara Penggunaan
1. Ketik nama guru di search bar
2. Pilih dari suggestions yang muncul
3. Lihat jadwal yang ter-highlight
